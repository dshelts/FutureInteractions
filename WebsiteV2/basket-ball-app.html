Bounce the Ball

This simulator is designed to give a basket ball realistic movement with respect to gravity, while giving the user the ability to hold and move the ball around the screen.

To view or download the file for this app visit the github link below.

Github file
For this simulator we used the Leap SDK and the Pygame library to make a simple user controlled animation. The balls movement is controlled using Pygame, while the hand image is controlled by the Leap Motions input based on the players hand position. When the hand is moved into contact with the ball the two will move as one with the images becoming one. To drop the ball at a new location the player simply moves their hand briefly upwards in a quick jerk, signaling the hand to release the ball.
